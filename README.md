JDK - Java 16

Framework - Spring Boot 2.4.4 

Server - Tomcat v9.0

Software Design Pattern - MVC 

Database – MySql Workbench 8.0(Spring Data Jpa)

Front-end: HTML,CSS,Thymeleaf
